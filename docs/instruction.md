# User Prompt

Saya ingin membuat library chart untuk web dan mobile. Saya ingin library ini dapat digunakan dengan mudah dan cepat. Saya ingin menulis library ini dengan typescript, dan hasilnya bisa digunakan baik di dalam kode typescript maupun javascript. Rencananya, library ini akan saya deliver ke npmjs, dengan nama 'aeva-chart', versi saat ini adalah '0.1.0'.

Library ini akan menggunakan chartjs versi terakhir. Selain itu juga menggunakan LitElement. Saya ingin menggunakan tsup sebagai building tools.

# Tugas

Tugasmu adalah bantu saya membuat scaffolding project ini untuk memudahkan saya. Struktur folder yang saya inginkan:
- src untuk source code dalam typescript, termasuk didalamnya css juga
- dist untuk hasil build menjadi javascript
- demo untuk file-file html yang menunjukkan cara kerja library (di dalam dist)
- file-file seperti package.json dan file-file konfigurasi untuk typescript dan tsup

Berikan plan langkah-langkahnya, dengan detil isi file yang bisa saya copy dan paste dan di mana harus saya letakkan.
